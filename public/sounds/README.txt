CUSTOM SOUND FILES
==================

This folder contains the game's sound effects.

GAME SOUNDS (played on host/board):
------------------------------------
intro.wav        - Welcome music when game starts
trial_tile.wav   - Thinking music for Trial Tile wagers
correct.wav      - Bell for correct answers
incorrect.wav    - Buzzer for wrong answers

PLAYER BUZZER OPTIONS:
----------------------
Players can choose from these custom buzzers when joining:
- angry_elephant
- baby_giggle
- bear_roar
- happy_alarm
- loud_fart
- surprised_chicken
- T_rex

Plus 8 built-in synthesized sounds (Classic, Bell, Horn, etc.)

ADDING MORE SOUNDS:
-------------------
Custom buzzers: Name files buzzer-custom-{name}.wav
Game sounds: Name files {effect}.wav or {effect}.mp3
  (intro, trial_tile, correct, incorrect, time_up, final_trivia)
