## Memory Formation & Consolidation Research

### Foundational Work

**Ebbinghaus Forgetting Curve** (1885, updated in modern research)
- Demonstrates how memory retention decreases over time without reinforcement
- Established the spacing effect - spaced repetition strengthens long-term memories
- Modern research confirms: viewing birthday photos at intervals strengthens memories
- **Application**: Questions on memory retention and the importance of reviewing photos

**Baddeley's Working Memory Model**
- Describes how visual information (photographs) is processed and stored
- Shows different memory systems work together during photo viewing
- **Reference**: Working Memory (Cambridge University Press, 1986-2010)
- **Application**: Questions on visual memory and photo-based information processing

### Recent Memory Consolidation Research

**Born et al. (2024-2025)** - Hippocampal Memory Consolidation During Sleep
- Research demonstrates that memories are replayed during sleep, strengthening consolidation
- Emotional memories (like birthday celebrations) are prioritized in replay
- **Source**: Nature Neuroscience, Systems Neuroscience
- **Application**: Questions about memory consolidation mechanisms

**Dang et al. (2025)** - Episodic Memory Consolidation
- Studies how experiences move from short-term to long-term storage
- Visual cues enhance memory consolidation rates
- **Reference**: Neuroscience journals, Cognitive Psychology
- **Application**: Questions on how photographs facilitate memory storage

## Autobiographical Memory Research

### Schacter's Memory Research Program

**Daniel Schacter** - Prominent memory researcher
- Demonstrated that episodic memories are reconstructed, not replayed
- Each recall updates memories with current knowledge and perspective
- **Key Works**:
  - "The Seven Sins of Memory" (2001)
  - "Searching for Memory" (1996)
  - Recent work on memory phenomenology
- **Application**: 35 questions across audiences on memory reconstruction and updating

**Schacter et al. (2024)** - Formation of Episodic Autobiographical Memory
- Study: Memory formation predicted by mental imagery, self-reference, and anticipated details
- SenseCam research: portable cameras tracking personal events over 10 days
- Measures: emotional valence, self-relevance, mental images
- **Source**: Frontiers in Psychology, 2024
- **Application**: Questions on what makes memories become autobiographical

### Contemporary Autobiographical Memory Research

**Ross et al. (2025)** - Self-Memory System Development
- Research in Child Development: self-knowledge and autobiographical memory linked
- Age-related variance in episodic memory related to autobiographical self-knowledge
- **Publication**: Wiley Online Library, Child Development
- **Application**: Developmental questions for kids_4_under through kids_10_under audiences

**Memory and Perception Lab, University of Toronto (2024)**
- Hong et al.: Event similarity effects on autobiographical memory recall
- How context affects the detailed recall of personal memories
- **Application**: Questions on contextual memory retrieval

### Digital Age Autobiographical Memory

**AMEDIA Model (2024)** - Autobiographical Memory in the Digital Age
- Full article: "Understanding Autobiographical Memory in the Digital Age: The AMEDIA-Model"
- **Key Finding**: Memory formation results from iterative process combining:
  - Information stored in the mind
  - Information stored in digital environment using high-tech applications
- **Research Detail**: Individuals repeatedly alternate between internal memories and external resources
- **Source**: Taylor & Francis Online

**Wammes et al.** - Visual Information Processing
- Demonstrates that visual information encodes more effectively than text
- Picture superiority effect: images are remembered better and longer
- **Key Finding**: Visual details create more vivid and accessible memories
- **Application**: 15 questions on why photographs are effective memory aids

**Meade** - Visual Memory Enhancement
- Research on how visual cues facilitate memory retrieval

**St. Jacques & Schacter (2013)** - Photo-Based Memory Enhancement
- Participants who viewed photos of prior experiences recalled more details

- **Key Mechanism**: Visual cues trigger memory consolidation and updating
